# yasfm #

Library for structure from motion.

# Related Publications #

* **[Image Matching for Dynamic Scenes](http://cmp.felk.cvut.cz/~srajefil/theses/filip-srajer-diploma-thesis.pdf)**, *F. Å rajer*, Diploma thesis. Czech Technical University in Prague. 2016.

# What do I need to build it? #

* Windows
* Microsoft Visual Studio 2013 (version 12) 64bit
* 5pt-6pt-Li-Hartley
* ceres-solver
* DevIL 
* Eigen
* FLANN
* gflags
* glew
* glog
* jhead
* SiftGPU
* suitesparse-metis-for-windows
* zlib

The 3rd party libraries are included in the repository. See COPYRIGHT.md file for their licenses.
